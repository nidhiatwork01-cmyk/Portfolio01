# Portfolio Website

Personal portfolio website built with HTML, CSS, and JavaScript.

## Overview

This repository contains a responsive portfolio site that presents:

- Profile and introduction
- Technical skills
- Featured projects
- Achievement highlights
- Education
- Contact information

The site is designed for clear project storytelling, strong visual hierarchy, and smooth interactions.

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome icons

## Project Structure

```
.
|- index.html
|- styles.css
|- script.js
|- hackathon-image.jpeg
|- speaking-1.jpg
|- speaking-2.jpg
|- speaking-3.jpg
```

## Run Locally

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

No build step is required.
